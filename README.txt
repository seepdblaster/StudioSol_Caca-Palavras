Feito por Bruno Rodrigues Paschoalino

Desenvolvimento realizado em Windows 7 através da IDE PyCharm utilizando: Python 3.7

Programa com objetivos de procurar e retornar a quantidade de palavras em sequência sem parar o funcionamento dentro de um caça palavras.

Como executar, para :

	Para windows, entre na pasta chamada dist e execute o arquivo para abrir no terminal o programa.
	
	Para linux ou macOS, abra o terminal na pasta que contém o programa e execute a linha $ py exercicioStudio.py

Para finalizar o programa digite 0(zero)
